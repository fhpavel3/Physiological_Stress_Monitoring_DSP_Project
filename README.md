## Overview

This project implements a DSP-based physiological stress monitoring system using a GSR sensor and Arduino UNO.

## Features

- Real-time GSR signal acquisition
- Gaussian FIR Low Pass Filtering
- Baseline calibration
- Stress variation analysis
- LED and buzzer indication

## Hardware

- Arduino UNO
- GSR Sensor
- LEDs
- Buzzer

## DSP Technique

Gaussian Window-based FIR Low Pass Filter is used for noise reduction and signal smoothing.
